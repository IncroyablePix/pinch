# Contributing

Thanks for improving Pinch.

## Development workflow

1. Install dependencies with `npm install`.
2. Run `npm run lint` and `npm run test` before opening a change.
3. Use `npm run build` to verify both browser targets still compile.
4. Use `npm run pack` when you need distributable zip archives.

## Pull requests

- Keep changes focused.
- Add or update tests when behavior changes.
- Update documentation when scripts or workflows change.

## Code style

This repository uses ESLint, Prettier, and a pre-commit hook powered by `lint-staged`.
